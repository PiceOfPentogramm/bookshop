## 🚀 Проекты

### [📚 Bookshop — Микросервисная e-commerce платформа](https://github.com/PiceOfPentogramm/bookshop)

> Продакшн-стиль онлайн-магазин книг как DevOps портфолио проект

- Спроектировал и развернул платформу из **7 микросервисов** на FastAPI, контейнеризированных Docker и оркестрированных Docker Compose
- Настроил **CI/CD пайплайн** через GitHub Actions: `lint → test → build → push → deploy`
- Реализовал мониторинг: сбор метрик **Prometheus** + визуализация в **Grafana**-дашбордах
- Настроил **nginx** как reverse proxy и API gateway с маршрутизацией между сервисами
- База данных **PostgreSQL** с миграциями через Alembic
